# 🔐 Login Page · Git Practice Project

A beautiful login page with light/dark mode to practice Git and GitHub workflows.

## 🚀 Features
- ✅ Clean, modern login interface
- ✅ Form validation
- ✅ Password visibility toggle
- ✅ "Remember me" with localStorage
- ✅ 🌙 Dark mode (added via feature branch)
- ✅ Keyboard shortcut: `Ctrl+Shift+D` to toggle dark mode
